Local Llama

Help/Instructions

Local Llama is a project that aims to make it easier to query local models. It is currently in development.

there is an acompaning script server.py which is a simple llama-cpp-python flask server

start the server in a terminal with python script.py

then you can query the server with the popup

you can click the check server button at the bottom to check if the server is up, green is up, Red is not

For more information, please visit the GitHub repository.
